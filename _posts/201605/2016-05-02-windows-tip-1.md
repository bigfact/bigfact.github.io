* 运行 cmd

* 输入命令

{% highlight command %}
> soundrecorder /file c:\app\ss.wma /duration 0000:00:05
{% endhighlight %}

* 说明
  * /file c:\app\ss.wma - 文件路径及文件名
  * /duration 0000:00:05 - 录制的时间 5s
