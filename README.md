# bigfact.github.io
bigfact.github.io

## 运行
> bundle exec jekyll serve

## Jekyll常用变量
* `site.time` 当前时间（跑jekyll这个命令的时间点）
* `site.pages` 所有 Pages 的清单
* `site.posts` 一个按照时间倒叙的所有 Posts 的清单
* `site.categories.CATEGORY` 所有的在 CATEGORY 类别下的帖子
* `site.tags.TAG` 所有的在 TAG 标签下的帖子
